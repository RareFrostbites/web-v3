Mga Tracking Technology at Mga Cookie

Ginagamit Namin ang mga Cookie at katulad na mga tracking technology para ma-track ang activity sa Aming Service at mag-store ng certain information. Ang mga Tracking technology na ginagamit ay mga beacon, mga tag, at mga script para ma-collect at ma-track ang information at para ma-improve at i-analyzw ang Ating Service. Ang mga Technology na ginamit Namin pwedeng may kasamang:

  • Mga Cookie o Mga Browser Cookie. Ang isang cookie ay isang small file na pini-place sa Device Mo. Pwede mong ituro ang browser Mo para ma-refuse ang lahat ng mga Cookie o para i-indicate kapag ang isang Cookie ay ipinapadala. However, 'pag 'di Mo tinatanggap ang mga Cookie, pwedeng hindi mo magagamit ang ilang mga part ng aming Service. Unless kung ina-adjust mo ang browser setting Mo para i-refuse nito ang mga Cookie, pwedeng gamitin ng aming Service ang mga Cookie.

  • Mga Flash Cookie. Ang mga certain feature ng aming Service ay pwedeng gumamit ng mga local stored objects (o mga Flash Cookie) para mag-collect at mag-store ng information about sa mga preference Mo o aktibidad Mo sa aming Service. Mga Flash Cookie ay hindi nai-manage ng parehong mga browser setting bilang mga ginagamit para sa mga Browser Cookie. Para sa more information kung paano mo matatanggal ang mga Flash Cookie, please basahin ang "Saan ko mababago ang mga setting para sa pagdi-disable, o pag-delete ng mga local shared objects?" na available sa https://helpx.adobe.com/flash-player/kb/disable-local-shared-objects-flash.html#main_Where_can_I_change_the_settings_for_disabling__or_deleting_local_shared_objects_

  • Mga Web Beacon. Ang mga certain section ng aming Service at ang aming mga email ay maaaring maglaman ng maliliit na electronic file na kilala bilang web beacon (tinukoy din bilang malinaw na mga gif, mga pixel tag, at mga single-pixel gif) na nagpe-permit sa Team, example, sa mga gumagamit na mga visitor sa mga page o na-open ang isang email at para sa iba pang related na mga website statistic (example, pagre-record ng isang certain na section at pag-verify ng system at integrity).

Ang mga Cookie ay pwedeng mga "Persistent" o "Session" Cookie. Mga Persistent Cookie ay magse-stay sa personal computer Mo o mobile device kapag ikaw ay offline, habang ang mga Session Cookie ay nai-delete sa lalong madaling panahon mo isinara ang web browser Mo. Alamin pa ang about sa mga cookie: Paggamit sa mga Cookie Free Privacy Policy .

Parehas Natin ginagamit ang mga Session at Persistent Cookie para sa mga purpose nai-set sa ibaba:

  • Necessary / Essential na mga Cookie

    Type: Mga Session Cookie

    Ina-administer ng: Tayo
